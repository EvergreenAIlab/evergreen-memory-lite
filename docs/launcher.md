# Local Launcher

The local launcher is a small Windows-first operation panel.

Run:

```text
Run Evergreen Memory Lite.bat
```

## Buttons

### Add files

Copies selected `.txt`, `.md`, `.docx`, and `.pdf` files into `inbox/`.

### Paste quick note

Saves a pasted note as a timestamped `.txt` file in `inbox/`.

### Run inbox

Runs local document intake and generates `output/latest`.

### Search memory

Searches `output/latest/search_index.sqlite`.

### Open latest dashboard

Opens `output/latest/dashboard.html`.

## Not a chat interface

The launcher is not a document chatbot. It does not use AI, RAG, or cloud services.
