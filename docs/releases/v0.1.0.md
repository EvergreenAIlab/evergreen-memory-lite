# v0.1.0 draft release notes

> Draft only. Version 0.1.0 has not been tagged or published.

Evergreen Memory Lite v0.1.0 establishes the initial public, Windows-first local demonstration.

## Included

- PowerShell-friendly setup and commands.
- Synthetic P0 sample data only.
- Dry-run-by-default runner that never deletes, moves, renames, overwrites, or changes source notes.
- Explicit P0-P4 privacy-tier model.
- Local Markdown cards, SQLite metadata registry, and audit events after explicit write approval.
- Pytest coverage and GitHub Actions CI on Windows.

## Current boundaries

This candidate is not intended for real sensitive data or production use. It has no cloud, API, or LLM integration and does not determine whether arbitrary content is safe. Use only deliberately created synthetic examples.
