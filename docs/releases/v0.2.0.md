# v0.2.0 release notes

v0.2.0 turns Evergreen Memory Lite from a safe local skeleton into a small useful household/small-office admin workflow.

## What changed

- Added the household admin value loop.
- Generates five local Markdown outputs:
  - `family_brief.md`
  - `action_list.md`
  - `timeline.md`
  - `source_index.md`
  - `risk_flags.md`
- Adds structured synthetic examples in English, Chinese, and Norwegian.
- Preserves UTF-8 content without translation.
- Supports fixed section heading aliases for actions, events, and risks.
- Keeps metadata keys deterministic and English-only.
- Keeps dry-run as the default behavior.
- Keeps source files unchanged.
- Keeps all generated outputs under ignored `output/`.

## Boundaries

- Synthetic P0 examples remain the default.
- Real sensitive data is not supported.
- No LLM, cloud API, RAG, vector database, web UI, bot, or background service is added.
- The parser is rule-based and does not infer arbitrary language meaning.
- Non-P0 notes are flagged and are not treated as supported safe input.

## Typical command

```powershell
.\.venv\Scripts\python.exe -m evergreen_memory_lite.runner `
  --input data\synthetic `
  --output output\household_admin `
  --household-admin `
  --write
```
